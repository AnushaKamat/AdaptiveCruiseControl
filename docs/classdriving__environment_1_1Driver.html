void Driver::update ( )

This is like a front end method. User can set the parameters using setters The update method depending upon the setting of ACC_on and CC_on switches , sets the operation mode for the car i.e; either REGULAR, CC or ACC when both CC_on and ACC_on is OFF(0), then car operates in REGULAR mode when ACC_on is set , car operates in ACC (Adaptive Cruise Control) mode when CC_on is set, car operates in CC(Cruise Control) Mode if both CC_on and ACC_on is set then ACC gets precedence and CC is internally is set to OFF Depending on the operation mode set by driver, the events are emited. if ACC, ACC status is emited, safety distance and Desired speed set by driver is also sent on the channels SafetyDistance and DesSpeed respectively In case of CC, CC status is emitted and desired_speed set by the driver is sent on DesDpeed channel In case of Regular mode, the driver presses acceleration pedal (accped) and the force applied is sent over Throttle channel
